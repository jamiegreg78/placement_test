  • 2.

    $string contains letters and numbers, the numbers aren't relevant to us, so we want to remove all of the numbers from the string. What approach would you take to achieve this?

    <?php

    $string = "u5AUPMMLWxfIAWBzhgtGeRyQs";

    $editedString = preg_replace('/[0-9]+/', '', $string);

    echo $editedString;

    This answer uses preg_replace to remove all characters matching a regular expression. The resulting string is then saved to a variable.

  • 3.

    $string has been encrypted using a basic encryption method. We know that all occurrences of the letters 'a, r, d' were replaced with 'c, m, e' respectively; we need to replace these letters with the respective letters. What approach would you take to achieve this?

    <?php

    $string = "u5AUPMMLWxfIAWBzhgtGeRyQs";

    $initialChars = array('c', 'm', 'e');

    $replaceChars = array('a', 'r', 'd');

    $decryptedString = str_replace($initialChars, $replaceChars, $string);

    echo $decryptedString;

    For this question, I used str_replace to search through the string and replace each occurence of a character with another. Since I used arrays, the function will replace each character with the respective decrpypted character in the other array. By swapping the arrays inputs around, you can perform the same encryption that the question asks for.

  • 4.

    We know that $string is a url, what we need to know is the domain name used in the url. What approach would you take to achieve this?

    <?php

    $url = "https://www.youtube.com/watch?v=8dyka2Yz9Tw";

    $domainName = parse_url($url, PHP_URL_HOST);

    echo $domainName;

    For this question I used parse_url to retrieve information about a url. Without specifying what data you want, the function will return an array with plenty of data about the url. By specifying that I wanted the host, I can get the domain name of the url easily.

  • 5.

    Lastly, there were 2 parameters passed in the url. We need to know the name and value of the 2 parameters. What approach would you take to achieve this?

    <?php

    $url = "http://www.example.com/bottle?bag=border&dog=small";

    $parsedUrl = parse_url($url);

    parse_str($parsedUrl['query'], $params);

    print_r($params);

    This question starts similarly to the last one, so I continued to use parse_url like before. This time, I store the whole array. I then use parse_str with the query data from the url which is then saved to the $params array. This breaks the query string into variables. I can now access both parameters by calling $params['bag'] and $params['dog'] respectively. On the other hand, I can print out said results in a readable format using print_r.
